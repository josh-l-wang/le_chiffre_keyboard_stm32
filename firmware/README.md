## RGB indicators

In `config.h`:

- You can control the brightness  of the leds using `RGB_INDICATOR_BRIGHTNESS (default 5 out of 255)`
- You can disable them entirely using `RGB_MODS_INDICATOR_ENABLE` and `RGB_LAYER_INDICATOR_ENABLE`
